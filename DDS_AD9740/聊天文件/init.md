# 项目目标
基于 EP4CE10E22C8N FPGA + STM32H723ZGT6 开发板，通过 FMC 总线实现一个 DDS 任意波形发生器，驱动 AD9740 DAC 输出 1Hz 正弦波（频率可调）。

# 硬件参数
- FPGA: EP4CE10E22C8N (10320 LE, 414Kbit RAM, 23 个 18x18 乘法器, 2 个 PLL)
- STM32: STM32H723ZGT6 (Cortex-M7 @ 550MHz, 双精度 FPU, FMC 总线)
- DAC: AD9740 (10-bit, 125MSPS 最大采样率)
- FMC 连接：16-bit 数据线 (FMC_D[15:0])，15-bit 地址线 (FMC_A[14:0])，控制线 FMC_NE1 (片选)、FMC_NOE (输出使能)、FMC_NWE (写使能)，无 NBL 信号。
- FPGA 时钟：外部晶振 50MHz，经 PLL 倍频至 125MHz 作为 DAC 采样时钟 (DACCLK)（目前还未连线）。
- STM32 时钟：内部振荡器，FMC 时钟配置为 110MHz。

# DDS 设计规格
- 输出频率范围：0.1Hz ~ 20MHz（工程推荐上限，理论可达 62.5MHz）
- 目标频率分辨率：1Hz（精度要求 ±0.01Hz）
- 相位累加器位宽：32 位（频率分辨率 = 125MHz / 2^32 ≈ 0.029Hz）
- 波形 RAM：深度 1024 (10-bit 地址)，每个点 10-bit 数据（匹配 AD9740   B0-B9）
- RAM 存储一个完整周期的正弦波（可由 STM32 初始化写入）（先实现正弦波，满幅值0-2047固定，暂时不写调节幅值的代码，，其他波形暂时不写）

# 寄存器地址映射（FPGA 端）
- 波形 RAM 区：地址偏移 0x0000 ~ 0x03FF（1024 个 16-bit 寄存器，实际只用低 14-bit）
- 频率控制字 (FTW)：地址偏移 0x0400 (32-bit，需分两次 16-bit 写入，低16位先写，高16位后写)
- 更新使能寄存器：地址偏移 0x0404 (bit0=1 时加载 FTW 到工作寄存器)
- 相位复位寄存器：地址偏移 0x0408 (写 1 清零相位累加器)
- DAC控制开启关闭寄存器：地址偏移0x040C（bit0=1 正常输出 bit0=0 强制DAC输出中间电位0x0200）

# 代码输出要求
- STM32 工程（Keil 或 CubeIDE）：目前已包含 FMC 初始化、寄存器写入读取功能经过测试已实现，需要FTW 计算、波形生成代码、波形写入寄存器代码 、DAC控制代码、测试主循环。（可以通过USART1进行调试，使用usart中断对stm32进行控制，控制内容为usart输出对应寄存器状态如DAC输出状态、波形加载是否正确）
- FPGA 工程（Quartus）：目前已包含 PLL 配置、FMC 从机接口、双端口 RAM、需要写入相位累加器、DAC 输出等。

# 注意事项
- FMC 异步读写时序：需要满足 FPGA 的建立/保持时间，使用 STM32 的 FMC 配置为 16-bit 异步模式。
- FPGA 端必须使用“影子寄存器”机制防止高/低 16 位写入时出现中间态频率跳变。
- DAC 输出数据需与 DACCLK 对齐，使用 FPGA 的 ODDR 原语输出。
